### UART PROGRAMMING NOTES

UART PROGRAMMING THROUGH THE **J3 PROGRAM HEADER** IS WORKING.

WORKING CONNECTIONS:

* USB-UART `TX` -> BOARD `ESP_RXD`
* USB-UART `RX` -> BOARD `ESP_TXD`
* USB-UART `GND` -> BOARD `GND`

CURRENT WORKING UPLOAD SEQUENCE:

1. HOLD **BOOT**
2. START UPLOAD
3. TAP **RESET** WHEN THE TOOL BEGINS CONNECTING
4. RELEASE **BOOT** AFTER CONNECTION
5. PRESS **RESET** AFTER UPLOAD TO RUN THE APPLICATION

## REV-A NOTES

### WHAT WORKED WELL

* CORE POWER-UP AND 3.3V REGULATION WERE GOOD
* ESP32-C6 BOOTED SUCCESSFULLY AND RAN APPLICATION CODE
* UART PROGRAMMING THROUGH THE J3 HEADER WORKED
* RGB LED, USER BUTTON, OLED, AND I2C SENSOR WERE ALL VERIFIED IN HARDWARE
* THE BOARD WAS ABLE TO RUN A COMBINED DEMO SHOWING LIVE TEMP / HUM DATA ON THE OLED

### WHAT TOOK TOO LONG

* TOO MUCH TIME WAS SPENT CHASING HOST-SIDE DRIVER / USB ISSUES BEFORE FALLING BACK TO THE KNOWN-GOOD UART PROGRAMMING PATH
* INITIAL BRING-UP WOULD HAVE MOVED FASTER WITH A MORE DELIBERATE STEP-BY-STEP VALIDATION PLAN FROM THE START
* LED / GPIO MAPPING HAD TO BE VERIFIED MANUALLY DURING TESTING INSTEAD OF BEING IMMEDIATELY OBVIOUS FROM A SINGLE BRING-UP REFERENCE

### WHAT MADE BRING-UP HARDER THAN IT NEEDED TO BE

* NATIVE USB ENUMERATION DID NOT COME UP CLEANLY DURING INITIAL TESTING
* PROGRAMMING CURRENTLY REQUIRES A MANUAL BOOT / RESET SEQUENCE THROUGH THE UART HEADER
* WITHOUT AUTO-RESET / AUTO-BOOT CONTROL, THE PROGRAM / TEST LOOP IS SLOWER AND MORE ERROR-PRONE
* A FEW SIGNAL / GPIO DETAILS NEEDED TO BE CONFIRMED IN HARDWARE INSTEAD OF BEING PRE-DOCUMENTED AS A BRING-UP MAP

### WHAT I WOULD NOT DO THE SAME WAY AGAIN

* I WOULD NOT RELY ON MEMORY OR SCATTERED NOTES FOR FIRST BRING-UP PIN MAPPING
* I WOULD CREATE A SMALL, DEDICATED BRING-UP TABLE FOR GPIO, I2C ADDRESSES, TEST POINTS, AND PROGRAMMING FLOW BEFORE THE BOARD ARRIVES
* I WOULD PLAN FOR A FASTER FIRST-FLASH / RESET FLOW SO THAT FIRMWARE ITERATION IS LESS FRAGILE
* I WOULD TREAT A WORKING UART PROGRAMMING PATH AS THE FIRST RECOVERY / VALIDATION TARGET IMMEDIATELY
